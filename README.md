# brainlabs-task
Simple campaign budget manager
1. Download ZIP from the code button drop down and unzip
2. Within the IDE / VS Code, open the folder and contained files and ensure all files are present within the viewfinder on the left
3. Ensure Docker for Desktop is running
4. Open a new terminal within the IDE and run the docker compose up command
5. Open a new browser and paste in http://localhost:8888, and the campaign budget manager app will appear
